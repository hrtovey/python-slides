Python is a computer language. It's a series of instructions that tells the computer to do something like: - displaying text - moving an image - asking the user for information It can be used for system administration, 3d animation and image editing, scientific computing, web development and game development.
Generally speaking, the computer reads each line of code from top to bottom, working out what action you want to take on each line, and then taking that action. This is called running or executing your code.
Python is an interpreted language. Your computer doesn't understand Python, so it needs something to interpret the code and convert it to machine code. This conversion happens at the time your code is executed. JavaScript, PHP and Ruby are also interpreted languages. In a compiled language (like C# and Java) the code is converted to machine code BEFORE it's excuted. Interpreted languages run a little slower than a compiled program, but you won't need to wait for your program to compile to run the code. If you need to quickly program a solution to some problem, python is a good choice.
- Being able to debug your own code is one of the most important skills you can have as a programmer. Practice debugging and this will speed up your Python knowledge gain.
